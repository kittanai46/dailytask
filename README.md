# About Application

DailyTask เป็นแอปพลิเคชันที่พัฒนาด้วย Flutter
เพื่อช่วยให้ผู้ใช้งานสามารถจัดการงานและบันทึกข้อมูลต่าง ๆ
ในชีวิตประจำวันได้อย่างเป็นระบบ

ผู้ใช้งานสามารถสร้างรายการงานที่ต้องทำ (To-Do List)
จดบันทึกข้อความสำคัญ บันทึกโน้ตส่วนตัว
จัดทำรายการซื้อของ รวมถึงบันทึกรายรับรายจ่าย
ภายในแอปพลิเคชันเดียว

แอปถูกออกแบบให้ใช้งานง่าย
ช่วยเพิ่มประสิทธิภาพในการวางแผนงาน การจัดการเวลา
และการติดตามข้อมูลสำคัญในชีวิตประจำวัน

# Features

## ✅ Task Management
สร้างและจัดการรายการงานที่ต้องทำ (To-Do List)

## 📝 Note Taking
จดบันทึกข้อความและข้อมูลสำคัญได้อย่างสะดวก

## 📒 Memo & Personal Notes
บันทึกไอเดีย ความคิด และข้อมูลส่วนตัว

## 🛒 Shopping List
สร้างรายการซื้อของและติดตามรายการที่ต้องการซื้อ

## 💰 Income & Expense Tracking
บันทึกรายรับรายจ่าย พร้อมสรุปข้อมูลทางการเงิน

## 🎨 Customizable Theme
ปรับแต่งสีและรูปแบบการแสดงผลของแอปได้

## 🏷 Custom Icons & Categories
กำหนดไอคอนและหมวดหมู่ให้กับงานหรือบันทึกแต่ละประเภท

## 📅 Daily Organization
ช่วยจัดระเบียบงานและกิจกรรมในแต่ละวัน

## 🔍 Easy Search & Management
ค้นหาและจัดการข้อมูลได้อย่างรวดเร็ว

## 📱 Modern User Interface
ออกแบบ UI ให้ใช้งานง่าย สวยงาม และเหมาะกับการใช้งานประจำวัน
